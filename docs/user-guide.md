# Pi Monitor – Brukerveiledning

## Hva er Pi Monitor?
Pi Monitor er et dashbord som viser sanntidsinformasjon om Raspberry Pi-ens ytelse. Du kan se hvor varm prosessoren er, hvor mye minne som brukes, og hvor full harddisken er.

## Slik åpner du dashbordet
1. Koble deg til samme nettverk som Raspberry Pi-en
2. Åpne en nettleser (Chrome, Safari, Firefox)
3. Gå til adressen: `https://pimonitor.local`
4. Nettleseren vil advare om at tilkoblingen ikke er privat – dette er normalt for et lokalt nettverk. Klikk **Avansert** og deretter **Fortsett**
5. Skriv inn brukernavn og passord når du blir bedt om det

## Hva viser dashbordet?

### CPU
Viser prosessorens temperatur i grader Celsius og hvor mye av prosessorkapasiteten som brukes. Høy temperatur over lengre tid kan indikere at Raspberry Pi-en jobber hardt.

### RAM
Viser hvor mye av arbeidsminnet som er i bruk. Dersom dette nærmer seg 100% kan systemet begynne å gå sakte.

### Disk
Viser hvor mye lagringsplass som er brukt og hvor mye som er ledig. Dersom disken er nesten full bør gamle filer slettes.

## Feilsøking
- **Siden laster ikke**: Sjekk at du er på samme nettverk som Raspberry Pi-en, eller bytt nettleser
- **Feil brukernavn/passord**: Kontakt systemadministrator for å få egen bruker
- **Dataene oppdateres ikke**: Prøv å laste siden på nytt
